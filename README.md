# Fase 1 (T01): Projeto Integrador: Radar Legislativo

### Primeiro projeto da POS. O intuito é usar os aprendizados em: git/github, python, pandas, IA e n8n.

---
### 1.⚠️ Criação de ambiente
- Cria o venv e adiciona o requirements.txt nele
- se usar o pip
```bash
python -m venv .venv
```

- ou se usar o uv:
```bash
uv venv
```

---
### 2.⚠️ Executação total no terminal
```bash
python app.py
```
 - Script `app.py` ja é um osquestraador simples para executar em sequencia utilizando subprocessos

### 3.⚠️ Gravar os dados
```bash
python app.py
```
 - na execução como há a obrigação de salvar num banco postgres é necessário tem um .env com todas as credenciais do banco
